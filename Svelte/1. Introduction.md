## Build new Project
### case 1
建立 Vite + Svelte 專案
SPA 專案，不含伺服器路由、SSR 等功能。
```zsh
yarn create vite svelte-app --template svelte-ts
```

路由套件
```zsh
yarn add svelte-spa-router
```

### case 2
建立的是 SvelteKit 專案
```zsh
npx sv create svelte-app
```

<script>
