# macOS install
## Install Flutter SDK
** ARM架構（ex M1）須先安裝rosetta2 **
```
mkdir 自訂資料夾
cd ~/自訂資料夾
unzip ~/flutter_macos_3.7.3-stable.zip
```

此命令僅為當前終端窗口設置PATH變量。要將Flutter永久添加到路徑，請參閱更新路徑。
```
export PATH="$PATH:`pwd`/flutter/bin"
```
